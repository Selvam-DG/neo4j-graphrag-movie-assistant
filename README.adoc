= Neo4j GraphRAG Portfolio Project

This repository contains a personal GraphRAG project built with Neo4j and Python.

The project demonstrates how Neo4j can be used with large language models to retrieve graph-based context and generate grounded responses.

It includes examples of:

* Vector retrieval
* Vector-based GraphRAG
* Vector and Cypher retrieval
* Text-to-Cypher generation
* Neo4j GraphRAG pipelines
* Environment validation and automated testing

== Features

=== Vector Retriever

Uses a Neo4j vector index to identify semantically relevant nodes based on a natural-language query.

=== Vector GraphRAG

Combines Neo4j vector retrieval with a large language model to generate answers grounded in retrieved graph data.

=== Vector and Cypher GraphRAG

Uses vector search together with Cypher queries to retrieve both semantically relevant content and connected graph information.

=== Text-to-Cypher GraphRAG

Converts natural-language questions into Cypher queries and uses the results as context for an LLM-generated response.

== Setup

Clone the repository and enter the project directory.

## [source,sh]

git clone https://github.com/Selvam-DG/genai-graphrag-portfolio.git
cd genai-graphrag-portfolio
---------------------------

Create a Python virtual environment.

## [source,sh]

python -m venv .venv
source .venv/bin/activate
-------------------------

Install the dependencies.

## [source,sh]

## pip install -r requirements.txt

Create a `.env` file using `.env.example` as a template.

## [source,sh]

## cp .env.example .env

Add your Neo4j and LLM credentials to `.env`.

## [source,env]

NEO4J_URI=neo4j+s://your-instance.databases.neo4j.io
NEO4J_USERNAME=neo4j
NEO4J_PASSWORD=your-password
OPENAI_API_KEY=your-api-key
---------------------------

Do not commit the `.env` file.

== Environment Check

Run the environment validation script.

## [source,sh]

## python graphrag_app/test_environment.py

== Running the Examples

Run the vector retriever example.

## [source,sh]

## python graphrag_app/vector_retriever.py

Run the vector GraphRAG example.

## [source,sh]

## python graphrag_app/vector_rag.py

Run the vector and Cypher GraphRAG example.

## [source,sh]

## python graphrag_app/vector_cypher_rag.py

Run the Text-to-Cypher example.

## [source,sh]

## python graphrag_app/text2cypher_rag.py

== Testing

Install pytest if it is not already installed.

## [source,sh]

## pip install pytest

Run the tests.

## [source,sh]

## pytest

== Security

Credentials are loaded from environment variables.

Never commit:

* `.env`
* API keys
* Neo4j passwords
* Database credentials
* Private certificates
